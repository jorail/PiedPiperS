PiedPiperS-Projekt im Internet

Alle Projekt­dokumente, Programm­codes, Anleitungen und Abbild­ungen finden sich im Internet bei https://github.com/jorail/PiedPiperS

Rufe die PiedPiperS-Projektseite bei GitHub über einen Browser mit Internet­verbindung auf. Dafür kann es nötig sein, zuvor die WLAN-Verbindung zum ESP32 zu trennen und eine neue Verbindung aufzubauen. Falls das gerade nicht geht, ist das Projekt später leicht wieder zu finden unter:
