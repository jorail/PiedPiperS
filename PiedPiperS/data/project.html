PiedPiperS-Projekt

Das Programm steuert eine Modell­bahn­lokomotive mit unabhängiger Strom­versorgung z.B. beim Einsatz im Freien. Für die Loksteuerung stehen folgende Möglichkeiten bereit:

a) Steuerung von einem Smartphone: ESP32 WLAN-Server für Kommandos der Nutzer auf einer Webseite im Browser und über das WebSocket-Protokoll:

PiedPiperS-Projekt

b) Steuerung per Pfeifton: Funktioniert mit einem angeschlos­senen Mikrofon und FFT-Analyse auf einem Teensy 4.0 Mikrocontroller:

PiedPiper-Projekt

c) Morse-Code über einen Schalter oder Berührungs­sensor hinter der Lokomotive

d) Zeicheneingabe über den seriellen Monitor (Verbindung mit der USB-Schnittstelle)

Der elektrische Aufbau ist auf dem ersten Wagen hinter der Lok untergebracht. Er besteht aus wenigen und einfachen Bauteilen. Die Loksteuerung kann mit geringen Kosten nachgebaut werden. Die Software ist kostenfreie und unter der copyleft Lizenz GNU GPL v3 veröffentlicht. Sie kann z.B. mit Arduino IDE via USB auf einen Mikor­controller übertragen werden.
